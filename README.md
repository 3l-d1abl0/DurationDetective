```
$ virtualenv -p python3 <your_env>

#### Install the required modules
```
$ cd DurationCheck
$ source <your_env>/bin/activate
$ pip3 install -r requirements.txt
```
#### Run the Script
```
$ python3 durationcheck.py --path "/path/to/your/folder"
